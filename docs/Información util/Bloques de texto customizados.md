## Tipos bloques de texto customizados

!!! note "Titulo"
    Esto es una nota, se escribe poniendo !!! note, y debajo el texto, si lo dejamos sin poner "" a la derecha de nota, no tendra titulo, sino pondra note.

??? note
    Podemos ocultar la nota, haciendo que tengar que pulsar si ponemos 3 ???.
